# Studio einrichten

Nach der Registrierung ist es Zeit, dein Studio zu konfigurieren. Diese Grundeinstellungen sind wichtig, da sie überall in der Plattform verwendet werden.

## Allgemeine Einstellungen

Navigiere zu **Einstellungen → Allgemein** um die Basisdaten deines Studios anzupassen.

### Studio-Informationen

| Feld | Beschreibung |
|------|--------------|
| Studio-Name | Der Name deines Studios (erscheint überall) |
| Beschreibung | Kurze Beschreibung für das öffentliche Profil |
| E-Mail | Kontakt-E-Mail für Kunden |
| Telefon | Kontakt-Telefonnummer |
| Adresse | Standort-Adresse |
| Website | Link zu deiner Website |

### Logo hochladen

1. Klicke auf das Logo-Feld
2. Wähle eine Bilddatei (PNG oder JPG, max. 2MB)
3. Das Logo wird automatisch zugeschnitten
4. Speichere die Änderungen

**Tipp**: Verwende ein quadratisches Logo für beste Darstellung.

## Buchungseinstellungen

Unter **Einstellungen → Buchungen** konfigurierst du, wie Buchungen funktionieren.

### Buchungsfenster

Das Buchungsfenster bestimmt, wie weit im Voraus Kunden buchen können.

- **Aktiviert**: Wenn aktiv, können Kunden nur bis X Tage/Wochen im Voraus buchen
- **Zeitraum**: Wähle 1 Tag bis 12 Monate
- **Beispiel**: "2 Wochen" bedeutet Kunden sehen nur Kurse der nächsten 14 Tage

### Buchungsstopp

Der Buchungsstopp verhindert Last-Minute-Buchungen.

- **Aktiviert**: Buchungen werden X Stunden/Tage vor Kursbeginn gesperrt
- **Zeitraum**: Typisch sind 2-24 Stunden
- **Beispiel**: "2 Stunden" bedeutet 2h vor Kursbeginn ist keine Buchung mehr möglich

### Credit-Ablauf-Modus

Wähle, wie Credits am Ablauftag verfallen:

- **Ende des Tages**: Credits sind bis 23:59 Uhr gültig
- **Exakte Uhrzeit**: Credits verfallen zur exakten Kaufzeit

## E-Mail-Einstellungen

Unter **Einstellungen → E-Mail-Templates** passt du die automatischen E-Mails an.

### Verfügbare Templates

| Template | Wird gesendet bei |
|----------|-------------------|
| Buchungsbestätigung | Erfolgreiche Buchung |
| Buchung ausstehend | Buchung wartet auf Bestätigung |
| Buchung abgelehnt | Admin lehnt Buchung ab |
| Buchung storniert | Stornierung |
| Warteliste | Platz auf Warteliste |
| Willkommen | Neue Registrierung |
| Passwort zurücksetzen | Passwort-Reset |
| Zahlungsbestätigung | Credit-Kauf erfolgreich |

### Template bearbeiten

1. Klicke auf ein Template
2. Bearbeite Betreff und Inhalt
3. Nutze Platzhalter wie `{name}`, `{kurs}`, `{datum}`
4. Vorschau ansehen
5. Speichern

## Branding

Passe das Erscheinungsbild an unter **Einstellungen → Allgemein**.

### Theme-Farben

Bookicorn passt sich automatisch an dein Logo an. Du kannst zusätzlich wählen:

- **Light Mode**: Helles Design (Standard)
- **Dark Mode**: Dunkles Design

### Kursleiter-Bezeichnung

Wähle, wie Trainer in deinem Studio genannt werden:

- Trainer
- Kursleiter
- Lehrer

Diese Bezeichnung erscheint überall in der Oberfläche.

## Standorte & Räume

Bevor du Kurse erstellst, lege mindestens einen Standort an.

### Standort erstellen

1. Gehe zu **Standorte**
2. Klicke **Neuer Standort**
3. Fülle aus:
   - Name (z.B. "Hauptstudio")
   - Adresse
   - Beschreibung (optional)
   - Öffnungszeiten (optional)
4. Speichern

### Raum hinzufügen

1. Öffne einen Standort
2. Klicke **Raum hinzufügen**
3. Fülle aus:
   - Name (z.B. "Yoga-Raum 1")
   - Kapazität (max. Teilnehmer)
   - Raumtyp (Training, Yoga, Meditation, etc.)
4. Speichern
